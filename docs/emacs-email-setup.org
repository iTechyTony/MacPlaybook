* Manage your email account with Emacs with Mu4e
** mu
   Install yaourt -S mu
   #+BEGIN_SRC sh
   yaourt -S mu
   #+END_SRC
** Launch
   #+BEGIN_SRC sh
   mu init index --maildir=~/Maildir
   #+END_SRC
** Add email addresses
   #+BEGIN_SRC sh
mu init --my-address=tony@arksolutions.it --my-address=tony.ampomah@netsells.co.uk --my-address=tony.ampomah.jw@gmail.com
   #+END_SRC
